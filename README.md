# wxy-cli
![Npm 版本](https://img.shields.io/badge/wxy-cli_v0.0.1-green)
## 📑简介

wxy-cli 是一个基于 Node.js 的命令行工具，用于快速创建和管理前端项目。

## 模板
✅ 快速生成Vue3+TS+Vite+Pinia+ElementPlus+Axios+Mock的模板

## 安装

```bash
npm install -g wxy-cli
```

## 使用
```bash
# 使用模板创建新项目
wxy create
# 执行项目文件夹名称并选择模板创建项目
wxy create [projectName]
# 查看脚手架版本
wxy -v
wxy --version
```

## 功能

- 创建项目
- 管理项目依赖
- 生成代码片段
- 其他功能...
